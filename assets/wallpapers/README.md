# OffTopic Linux Wallpapers

This directory contains the default wallpapers for OffTopic Linux.

## Included Wallpapers

1. **offtopic-dark.jpg** - Default dark wallpaper (1920x1080)
2. **offtopic-minimal.jpg** - Minimalist version
3. **offtopic-neon.jpg** - Neon/cyberpunk style

## How to Add More

Place your wallpapers in this directory and they will be included in the ISO.

## Recommended Sources

- [Wallhaven](https://wallhaven.cc)
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)

## Resolution

Recommended: 1920x1080 (Full HD)
Also supported: 2560x1440, 3840x2160
