# Kali Openbox Edition

Lightweight Kali Linux with Openbox desktop environment and native Nvidia support.

## Features

- **Lightweight**: Openbox window manager instead of heavy desktop environments
- **Nvidia Ready**: Native Nvidia driver and CUDA support built-in
- **Mabox-inspired**: Modern dark theming, tint2 panel, customizable menu
- **Pentester-focused**: Pre-configured with essential security tools

## Quick Start

### Download ISO
Pre-built ISOs available in [Releases](https://github.com/yourusername/kali-openbox/releases)

### Build from Source

```bash
# Clone the repo
git clone https://github.com/yourusername/kali-openbox.git
cd kali-openbox

# Run the build script
sudo ./build.sh
```

## Requirements

- 40GB+ disk space
- 4GB+ RAM
- Nvidia GPU (optional, for driver support)

## What's Included

### Desktop
- Openbox window manager
- Tint2 panel
- Nitrogen wallpaper manager
- Rofi app launcher
- Terminator terminal

### Tools
- Essential Kali penetration testing tools
- Pre-configured .zshrc with useful aliases
- Custom Openbox menu with categorized tools
